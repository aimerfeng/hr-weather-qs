## [1.0.0] - 2024-12-26

### 新增
- ✨ 完整的终端版本智能助手
- 🌤️ 实时天气查询功能（支持任意城市）
- 📊 天气查询历史记录（最多10条）
- 💼 深度职业规划服务（结构化面试 + 个性化报告）
- 🔄 流式输出支持（实时显示 AI 响应）
- ⚙️ 多 AI 提供商支持（OpenAI / DeepSeek / 通义千问）
- 🎨 彩色终端界面（使用 Rich 库）
- 📦 一键安装脚本（Windows / Linux / Mac）
- 📖 完整的使用文档和分发指南
- 🧪 单元测试和属性测试覆盖

### 功能特性

#### 天气查询
- 查询任意城市的实时天气
- 获取未来5天天气预报
- 自动保存查询历史
- 智能城市名称纠正
- 好天气/坏天气区分

#### 职业规划
- 6步结构化面试流程
- 个性化职业规划报告生成
- 包含9个标准化章节：
  - 执行摘要
  - 个人档案分析
  - 职业方向推荐（至少3个）
  - 行业分析
  - 技能差距分析
  - 学习路径（免费/付费资源）
  - 技术栈推荐
  - 时间线和里程碑
  - 优先级行动项

#### 配置管理
- 交互式配置向导
- 支持预设提供商和自定义配置
- 配置持久化（JSON 格式）
- API 连接测试
- API Key 掩码显示

#### 用户体验
- 流式输出（逐字显示响应）
- 彩色终端界面
- 进度条显示
- 特殊命令支持（/help, /clear, /config, /history, /cancel）
- 友好的错误提示
- AI 身份过滤（不暴露 AI 身份）

### 技术实现
- Python 3.8+ 支持
- OpenAI SDK 集成
- Pydantic 数据验证
- Rich 终端美化
- asyncio 异步处理
- wttr.in 天气 API
- 模块化设计
- 完整的类型注解

## 版本说明

### 版本号格式
- **主版本号**：不兼容的 API 修改
- **次版本号**：向下兼容的功能性新增
- **修订号**：向下兼容的问题修正

### 更新类型
- **新增** - 新功能
- **变更** - 现有功能的变更
- **弃用** - 即将移除的功能
- **移除** - 已移除的功能
- **修复** - 问题修复
- **安全** - 安全相关的修复
